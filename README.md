## Docker Cheat Sheet

#### Check Docker version
```
docker --version
```

#### List all local images
```
docker image ls
```

#### List all local images
```
docker image ls
```

#### Run a Docker image
```
docker run [image]
```

#### List all containers
```
docker ps
```

#### List all running containers
```
docker ps --all
```
